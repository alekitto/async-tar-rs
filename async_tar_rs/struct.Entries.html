pub fn raw(self, raw: bool) -> Entries<'a, R>

Indicates whether this iterator will return raw entries or not.

If the raw list of entries are returned, then no preprocessing happens on account of this library, for example taking into account GNU long name or long link archive members. Raw iteration is disabled by default.
